# Financial Dashboard App

A web-based financial dashboard built with Next.js that demonstrates authentication-protected routes and basic CRUD workflows for managing financial records.

## Overview
This project models a simple internal dashboard experience where authenticated users can view, create, edit, and delete invoice data through a structured UI. It focuses on common patterns used in admin and reporting tools, including protected routes, form-based data entry, and dashboard layouts.

The project was initially based on a Next.js App Router starter and extended with custom logic and UI to support financial workflows.

## Features
- Authentication-protected dashboard
- CRUD interface for managing invoices
- Modular, component-based UI
- Server and client components using Next.js App Router
- Routing and loading patterns suitable for scalable web apps

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Node.js

## Status
This project is a functional prototype intended to demonstrate frontend architecture and dashboard patterns rather than a production deployment.

## Getting Started
### Prerequisites
- Node.js
- npm

### Installation
```bash
git clone https://github.com/natalieday/financial-dashboard-application.git
npm install
npm run dev
```
